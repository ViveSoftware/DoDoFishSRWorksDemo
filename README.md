# DoDoFishSRWorksDemo
This project use SRWorks SDK to demonstrate 3 AR games, developer can save time by learning from this demo to make AR illusion and interaction.

4. Import DOTween (HOTween v2)

5. Open Assets\Game\XRDemo.unity
